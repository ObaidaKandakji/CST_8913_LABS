# Cloud Resource Descriptions and Comparison Table:

| #  | Description | AWS (Service Name) | Azure (Service Name) | Google Cloud (Service Name) | Differences | Similarities |
|----|-------------|--------------------|----------------------|-----------------------------|-------------|--------------|
| 1  | A compute service that provides scalable virtual machines for running applications | Amazon Elastic Compute Cloud | Azure Virtual Machines | Google Compute Engine |  | All give you virtual machines you can start and stop |
| 2  | An object storage service used to store and retrieve data, commonly used for backups and static website content | Amazon Simple Storage Service | Azure Blob Storage | Google Cloud Storage |  | All are object storage with versioning and lifecycle tools |
| 3  | A managed relational database service that supports multiple database engines like MySQL, PostgreSQL, and SQL Server | Amazon Relational Database Service | Azure SQL Database and Azure Database for MySQL and Azure Database for PostgreSQL | Google Cloud SQL | Azure splits services by engine while the others group engines under one service | Managed relational databases with backups and scaling |
| 4  | A serverless compute service that allows you to run code in response to events without provisioning or managing servers | AWS Lambda | Azure Functions | Google Cloud Functions |  | Run small functions on events without managing servers |
| 5  | A virtual private network service that allows you to create isolated networks within the cloud provider's infrastructure | Amazon Virtual Private Cloud | Azure Virtual Network | Google Virtual Private Cloud |  | Private networks with subnets and routing |
| 6  | A content delivery network service that delivers data, videos, applications, and APIs to customers around the world with low latency | Amazon CloudFront | Azure Content Delivery Network | Google Cloud CDN |  | Global caching to speed up content |
| 7  | A managed NoSQL database service designed for low-latency, high-scale applications | Amazon DynamoDB | Azure Cosmos DB | Google Cloud Bigtable | Data models are different | Low latency NoSQL at big scale |
| 8  | A block storage service for use with virtual machines, offering persistent storage for data | Amazon Elastic Block Store | Azure Managed Disks | Google Persistent Disk |  | Persistent block storage for virtual machines |
| 9  | A managed container orchestration service based on Kubernetes | Amazon Elastic Kubernetes Service | Azure Kubernetes Service | Google Kubernetes Engine | | Managed Kubernetes control planes and nodes |
| 10 | A service for managing user access and encryption keys to secure cloud resources | AWS Key Management Service | Azure Key Vault | Google Cloud Key Management Service | | Key storage and rotation with fine grained controls |
| 11 | A platform that automates application deployment and scaling without needing to manage infrastructure | AWS Elastic Beanstalk | Azure App Service | Google App Engine | Runtimes and deploy flow are a bit different | PaaS for web apps without managing servers |
| 12 | A service that provides monitoring and logging of applications and infrastructure, offering insights into resource usage and performance | Amazon CloudWatch | Azure Monitor | Google Cloud Monitoring and Google Cloud Logging | Google splits monitoring and logging into two services | Metrics, logs, alerts, and dashboards |
| 13 | A domain name system service that routes traffic globally and translates domain names to IP addresses | Amazon Route 53 | Azure DNS | Google Cloud DNS | Route 53 also includes health checks and traffic policies | Authoritative DNS hosting with APIs |
| 14 | A load balancing service that distributes incoming network traffic across multiple targets, improving application availability | Amazon Elastic Load Balancing | Azure Load Balancer | Google Cloud Load Balancing | | Distribute traffic to keep apps available |
| 15 | A service that automatically scales your cloud infrastructure based on demand, ensuring resources are available as needed | AWS Auto Scaling | Azure Virtual Machine Scale Sets | Google Compute Engine Autoscaler | Azure uses scale sets and Google uses instance groups | Scale out and in automatically based on policies |
| 16 | A message queuing service that enables applications to send and receive messages between different components | Amazon Simple Queue Service | Azure Service Bus | Google Cloud Pub Sub | Google is publish and subscribe | Decouple components and buffer messages |
| 17 | A managed real time data streaming service that collects and processes large amounts of data from various sources | Amazon Kinesis Data Streams | Azure Event Hubs | Google Cloud Pub Sub | Ordering and protocol features are different a bit | Real time streaming ingestion at scale |
| 18 | A fully managed, highly scalable data warehouse service optimized for analytics and large scale queries | Amazon Redshift | Azure Synapse Analytics | Google BigQuery | BigQuery is serverless | SQL data warehouses for analytics |
| 19 | A service that automates the execution of workflows and allows the integration of different cloud services in a sequence of steps | AWS Step Functions | Azure Logic Apps | Google Cloud Workflows | Syntax and limits are different | Connect services into workflows |
| 20 | A service that integrates multiple data sources and enables data migration, transformation, and movement across platforms | AWS Glue | Azure Data Factory | Google Cloud Data Fusion | Glue uses serverless Spark and Data Fusion is based on CDAP | ETL and data movement between many sources |
| 21 | A data catalog and governance service that helps manage metadata across your data estate, supporting compliance and security | AWS Glue Data Catalog | Microsoft Purview | Google Cloud Dataplex | Purview and Dataplex include broader governance while Glue Catalog is more about metadata | Central catalog and lineage tools |
| 22 | A set of machine learning and AI services that provide pre built models, APIs, and tools for developers to easily implement AI in their apps | Amazon Bedrock | Azure AI Services | Google Vertex AI | Model choices and packaging differ | Prebuilt AI APIs and training tools |
| 23 | A service that allows you to define and deploy infrastructure using code, automating the management of cloud resources | AWS CloudFormation | Azure Resource Manager and Bicep | Google Cloud Deployment Manager | Azure also has Bicep and Google often uses Config Connector too | Define infrastructure as code and deploy it |
| 24 | A fully managed CI CD service that automates the building, testing, and deployment of applications to production environments | AWS CodePipeline | Azure Pipelines | Google Cloud Build and Google Cloud Deploy | Google splits build and deploy into two services | Automated build test and deploy pipelines |

| 26 | A backup and disaster recovery service that helps to protect your data by creating backups and replicas of your cloud resources | AWS Backup | Azure Backup | Google Cloud Backup and DR Service | Sources and features are different by cloud | Central policy based backups and restores |
| 27 | A service designed for big data analytics, allowing organizations to store, process, and analyze large datasets in real time | Amazon Elastic MapReduce | Azure HDInsight | Google Cloud Dataproc | Engine choices are different | Managed Hadoop and Spark ecosystems |
| 28 | A file storage service for storing and sharing files with users, typically used in shared file systems across applications | Amazon Elastic File System | Azure Files | Google Cloud Filestore | Protocols and performance tiers are different | Shared file storage for applications |
| 29 | A service that helps you transcode, process, and stream media content such as video and audio | AWS Elemental MediaConvert | Azure Media Services | Google Cloud Transcoder API |  | Transcode and stream media content |
| 30 | A real time communication service used for sending notifications, emails, and text messages to users and devices | Amazon Simple Notification Service and Amazon Simple Email Service and Amazon Pinpoint | Azure Communication Services | Firebase Cloud Messaging | Google does not include first party email or SMS here | Push style notifications to apps and devices |
